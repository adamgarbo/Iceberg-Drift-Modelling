# Iceberg Drift Modelling

## Description

### Data Processing
* Code relating to the processing of iceberg beacon tracjectories extracted from the recently compiled iceberg tracking beacon database. 

### Environmental Input Data
* Code pertaining to the processing of environmental forcing data used in the validation study of the NAIS iceberg drift model.
  * E.g. Scripts to rotate CECOM and RIOPS modelled ocean current data. 

### Figures
* All related code used to produce figures contained in the "Validation of the North American Ice Service Iceberg Drift Model" thesis.
